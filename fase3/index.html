HTML y CSS

HTML5 no se limita a ser un lenguaje de etiquetas HTML que sólo permiten definir elementos básicos, sino que combina nuevas etiquetas de HTML, propiedadees CSS3, Javascript y algunas otras tecnilogías. todas ellas suponen una actualización de gran potencia al conjunto de herramientas ya existente, y con él se pueden crear páginas web más sofisticadas y útiles.
En estos objetos virtuales de aprendizaje se busca brindar un conocimiento sobre temas relacionados con diseño de sitios web, como lo son las etiquetas de HTML y los atributos y valores que pueden tomar.
